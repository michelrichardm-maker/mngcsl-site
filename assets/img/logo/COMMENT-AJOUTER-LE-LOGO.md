# Logo du site MNG.CSL — mode d'emploi

Aujourd'hui, le logo dans l'en-tête est un **logotype composé** (une étoile SVG
verte + le texte « MNG ✦ CSL » + « Conseils & Associés »), directement dans le
code. Si tu as un **vrai logo graphique**, dépose-le ici et je l'intègre.

## Où
`assets/img/logo/`  ← ce dossier

## Fichiers utiles (dépose ce que tu as)
- `logo.svg` — logo principal (en-tête), idéalement **horizontal**, version couleur.
- `logo-blanc.svg` — variante **blanche/claire** pour les fonds foncés (pied de page, bandeaux marine). Optionnel mais recommandé.
- `logo-icone.svg` — juste le symbole/monogramme (sans texte), carré. Optionnel.

## Format
- **SVG** de préférence (net et léger). Sinon **PNG transparent**, hauteur ≥ 200 px.
- Fond transparent, marges minimales.

## Aussi remplaçables (icônes du navigateur / partage) — dossier parent `assets/img/`
- `favicon.svg` — petite icône d'onglet (carré, lisible à 16–32 px).
- `apple-touch-icon.png` — icône iOS (180×180 px).
- `og-image.jpg` — image d'aperçu lors des partages (1200×630 px).

## Ensuite
Préviens-moi : je remplace le logotype texte par ton image dans l'en-tête et le
pied de page (avec la variante claire sur fond foncé), en gardant l'accessibilité
(texte alternatif) et le lien vers l'accueil.
